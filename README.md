# 🏃‍♀️ FitPulse – Health Anomaly Detection from Fitness Devices

## 📌 Overview

FitPulse is a data-driven health monitoring application that detects anomalies in fitness data collected from wearable devices. It helps identify unusual patterns in health metrics like heart rate, steps, and activity levels.

---

## 🚀 Features

* 📊 Real-time fitness data analysis
* ⚠️ Anomaly detection using statistical & ML techniques
* 📈 Data visualization for better insights
* 🧠 Feature extraction using tsfresh
* 🌐 Interactive dashboard built with Streamlit

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries Used:** Pandas, NumPy, Matplotlib
* **Feature Engineering:** tsfresh
* **Visualization:** Matplotlib / Seaborn
* **Frontend:** Streamlit
* **IDE:** VS Code

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/FitPulse.git
cd FitPulse
```

### 2️⃣ Create virtual environment

```
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the application

```
streamlit run app.py
```

## 📊 How It Works

1. Collect fitness data from devices
2. Preprocess and clean the dataset
3. Extract meaningful features using tsfresh
4. Apply anomaly detection techniques
5. Visualize results using Streamlit dashboard
